# Security Policy

## Supported Versions

Minor releases will receive security updates and fixes until the next minor or
major release.

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2.0 | :x:                |

## Reporting a Vulnerability

Report a vulnerability to thek4n@yandex.ru
